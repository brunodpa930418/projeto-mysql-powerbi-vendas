# 📊 Projeto MySQL + Power BI - Sistema de Vendas

## 📌 Sobre o projeto
Este projeto simula um sistema simples de vendas, com banco de dados em MySQL e análise de dados no Power BI.

---

## ⚙️ Tecnologias usadas
- MySQL
- Power BI
- SQL (DDL e DML)

---

## 🗂️ Estrutura do projeto
- sql/ → scripts do banco de dados
- powerbi/ → dashboard em Power BI
- docs/ → documentação e imagens

---

## 🧱 Modelo do banco
O banco contém tabelas como:
- clientes
- produtos
- vendas

---

## 📊 Objetivo da análise
- Entender perfil de clientes
- Analisar vendas por produto
- Identificar desempenho por cidade

---

## 🚀 Aprendizados
- Criação de banco de dados relacional
- Consultas SQL (SELECT, GROUP BY, JOIN)
- Integração com Power BI
